# MediSmart — Medication Reminder & Management System

### Developed by: Uzma Tasnim Viva | IT Capstone Part II | IOU 2026

---

## About
MediSmart is a cross-platform medication reminder and management 
system that helps patients improve medication adherence through 
timely reminders, adherence tracking, analytics, and reporting.

## Features
- ✅ User Authentication (Register, Login, Password Reset)
- ✅ Medication Management (Add, Edit, Delete, Search)
- ✅ Daily Medication Reminders (Notifications)
- ✅ Adherence Tracking (Mark Taken / Missed)
- ✅ Reports & Analytics (Charts + PDF/CSV Export)
- ✅ Caregiver Patient Monitoring
- ✅ Admin Portal (Users, Logs, Settings)
- ✅ Responsive Web Application

## Tech Stack
- Flutter 3.27.1 + Dart 3.6.0
- Firebase Authentication + Cloud Firestore
- Provider State Management
- Flutter Local Notifications
- FL Chart
